# Niezbędne oprogramowanie

Tower nie potrzebuje żadnych zewnętrznych bibliotek do poprawnego działania. Jedynym potrzebnym zewnętrznym komponentem jest baza daych MongoDB

# Instalacja

Tower jest dystrybuowany w postaci spakowanej. Instalacja polega na rozpakowaniu otrzymanego archiwum do dowolnego katalogu systemu operacyjnego.
